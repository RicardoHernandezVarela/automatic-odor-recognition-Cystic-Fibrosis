Reconocimiento de patrones aplicado a muestras de aliento exhalado de voluntarios para diferenciar entre 
sujetos con fibrosis y un grupo de control.

Pruebas de clasificación con algoritmos disponibles en la librería de scikit-learn:

KNN
NAIVE BAYES
SVC
RANDOM FOREST

Patients with Chronic Pulmonary Infections
